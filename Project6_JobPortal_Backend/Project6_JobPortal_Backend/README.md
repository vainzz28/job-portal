# Project 6 - Job Portal Backend API (Advanced)

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Role Based Access (Candidate / Recruiter / Admin)

## Features
- Register/Login with JWT
- Recruiter can create jobs
- Candidate can apply jobs
- View applications
- Update application status

## Setup
### 1) Install dependencies
npm install

### 2) Create .env file
Copy `.env.example` and rename to `.env`

### 3) Start MongoDB
Make sure MongoDB is running locally.

### 4) Run server
npm start

Server runs at:
http://localhost:5000

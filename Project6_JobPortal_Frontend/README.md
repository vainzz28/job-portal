# Project 6 - Job Portal Frontend (React)

## Features
- Register/Login with JWT
- Browse jobs with search & filter
- Candidate can apply jobs
- View my applications
- Recruiter can post jobs

## Setup
1. Extract ZIP
2. Open terminal inside folder
3. Install:
   npm install
4. Start:
   npm start

Frontend runs at:
http://localhost:3000

Backend must run at:
http://localhost:5000
